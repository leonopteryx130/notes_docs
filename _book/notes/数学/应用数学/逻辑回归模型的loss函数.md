# 逻辑回归模型的loss函数

### 统计学视角

**思想**：找到一组参数 w，使得当前观测到的所有样本出现的概率（似然性）最大。

假设样本独立同分布，所有样本的联合概率（似然函数 $$L$$）为各个样本概率的乘积：

$$
L(w) = \prod_{i=1}^{N} P(y_i \mid x_i) = \prod_{i=1}^{N} \left[\hat{y}_i^{y_i} \cdot (1 - \hat{y}_i)^{1 - y_i}\right]
$$

为了计算方便（连乘变连加），取对数，得到对数似然函数：

$$
\ell(w) = \ln L(w) = \sum_{i=1}^{N} \left[y_i \ln \hat{y}_i + (1 - y_i)\ln(1 - \hat{y}_i)\right]
$$

机器学习中的习惯是最小化损失函数，所以我们取负号，得到负对数似然（NLL）：

$$
J(w) = -\sum_{i=1}^{N} \left[y_i \ln \hat{y}_i + (1 - y_i)\ln(1 - \hat{y}_i)\right]
$$

### 信息论视角

**思想**：衡量两个概率分布之间的差异。真实分布 $$P$$（即实际的 $$y$$）和预测分布 $$Q$$（即预测的 $$\hat{y}$$）。

对于二分类问题，每个样本的真实分布是 $$[y_i, 1 - y_i]$$，预测分布是 $$[\hat{y}_i, 1 - \hat{y}_i]$$。

交叉熵的定义为：

$$
H(P, Q) = -\sum_x P(x) \log Q(x)
$$

将类别代入（只有两个类别 0 和 1）：

$$
H(P, Q) = -\left[y_i \log \hat{y}_i + (1 - y_i)\log(1 - \hat{y}_i)\right]
$$

因为交叉熵越小，代表两个分布越接近，预测越准。将所有样本的交叉熵加起来，得到总损失：

$$
J(w) = -\sum_{i=1}^{N} \left[y_i \log \hat{y}_i + (1 - y_i)\log(1 - \hat{y}_i)\right]
$$

### 对比与结论

上面两节里的公式一模一样；当标签为离散硬标签时，**交叉熵 = 负的对数似然**。

- **MLE**：问的是「这组参数生成数据的概率有多大？」——越大越好。机器学习习惯最小化损失，所以对对数似然取负号来训练。
- **交叉熵**：问的是「预测分布离真实分布有多远？」——越小越好，直接最小化即可。
