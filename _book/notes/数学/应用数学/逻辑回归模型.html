逻辑回归模型

逻辑回归模型

理论基础：线性回归模型

线性回归用特征的线性组合直接给出预测值。对第 iii 个样本，模型为

y^i=w⊤xi+b,\hat{y}_i = \mathbf{w}^\top \mathbf{x}_i + b,​y​^​​​i​​=w​⊤​​x​i​​+b,

其中 xi\mathbf{x}_ix​i​​ 为特征向量，w\mathbf{w}w 为权重，bbb 为偏置。训练时在 loss 函数（如 MSE）下通过梯度下降求出 w\mathbf{w}w 和 bbb，使预测值 y^i\hat{y}_i​y​^​​​i​​ 尽量接近真实标签 yiy_iy​i​​。具体定义、loss 与梯度推导见 线性回归模型。

逻辑回归与线性回归共享同一套「线性打分」结构：都是先算一个实数

zi=w⊤xi+b,z_i = \mathbf{w}^\top \mathbf{x}_i + b,z​i​​=w​⊤​​x​i​​+b,

再把这个 ziz_iz​i​​ 解释成某种「预测结果」。

什么是逻辑回归模型？

逻辑回归（Logistic Regression） 主要用来做 二分类：给定特征 x\mathbf{x}x，估计样本属于正类（y=1y=1y=1）的概率 ppp。

为什么不能直接把线性回归用于二分类？ 二分类标签常记为 y∈0,1y \in 0, 1y∈0,1，我们真正想要的是「属于正类的概率」p∈[0,1]p \in [0, 1]p∈[0,1]。线性组合 w⊤x+b\mathbf{w}^\top \mathbf{x} + bw​⊤​​x+b 可以取任意实数，既可能小于 000，也可能大于 111，无法直接当作概率；若仍用 MSE 去拟合 0/10/10/1 标签，拟合面也不如专门的概率模型合理。逻辑回归的做法是：保留线性回归的打分形式，再增加一步把实数映射到 [0,1][0, 1][0,1] 的变换——这一步通常借助 概率，几率，对数几率 中的 sigmoid 与 logit。

对第 iii 个样本，先算线性打分 zi=w⊤xi+bz_i = \mathbf{w}^\top \mathbf{x}_i + bz​i​​=w​⊤​​x​i​​+b，再用 sigmoid 函数把 ziz_iz​i​​ 压到 [0,1][0, 1][0,1]，得到属于正类的预测概率：

p^i=sigmoid(zi)=11+e−zi.\hat{p}_i = \mathrm{sigmoid}(z_i) = \frac{1}{1 + e^{-z_i}}.​p​^​​​i​​=sigmoid(z​i​​)=​1+e​−z​i​​​​​​1​​.

等价地，模型在对数几率（logit）空间做线性回归：

logit(p^i)=lnp^i1−p^i=zi=w⊤xi+b.\text{logit}(\hat{p}_i) = \ln\frac{\hat{p}_i}{1 - \hat{p}_i} = z_i = \mathbf{w}^\top \mathbf{x}_i + b.logit(​p​^​​​i​​)=ln​1−​p​^​​​i​​​​​p​^​​​i​​​​=z​i​​=w​⊤​​x​i​​+b.

因此逻辑回归的全流程可以概括为：线性打分 → sigmoid → 概率。

loss 函数推导

线性回归的 MSE 为（见 线性回归模型）：

L(w,b)=1m∑i=1m(yi−y^i)2,y^i=w⊤xi+b.\mathcal{L}(\mathbf{w}, b) = \frac{1}{m}\sum_{i=1}^{m} \bigl(y_i - \hat{y}_i\bigr)^2, \qquad \hat{y}_i = \mathbf{w}^\top \mathbf{x}_i + b.L(w,b)=​m​​1​​∑​i=1​m​​(y​i​​−​y​^​​​i​​)​2​​,​y​^​​​i​​=w​⊤​​x​i​​+b.

逻辑回归保留同样的线性打分，但把 y^i\hat{y}_i​y​^​​​i​​ 换成 对数几率 logit(p^i)\text{logit}(\hat{p}_i)logit(​p​^​​​i​​)：

logit(p^i)=lnp^i1−p^i=w⊤xi+b,p^i=11+e−(w⊤xi+b).\text{logit}(\hat{p}_i) = \ln\frac{\hat{p}_i}{1 - \hat{p}_i} = \mathbf{w}^\top \mathbf{x}_i + b, \qquad \hat{p}_i = \frac{1}{1 + e^{-(\mathbf{w}^\top \mathbf{x}_i + b)}}.logit(​p​^​​​i​​)=ln​1−​p​^​​​i​​​​​p​^​​​i​​​​=w​⊤​​x​i​​+b,​p​^​​​i​​=​1+e​−(w​⊤​​x​i​​+b)​​​​1​​.

将 p^i\hat{p}_i​p​^​​​i​​ 代入二分类的 交叉熵损失（也称 对数损失 / log loss），即得逻辑回归的 loss：

L(w,b)=−1m∑i=1m[yilnp^i+(1−yi)ln(1−p^i)].\mathcal{L}(\mathbf{w}, b) = -\frac{1}{m}\sum_{i=1}^{m} \Bigl[y_i \ln \hat{p}_i + (1 - y_i)\ln(1 - \hat{p}_i)\Bigr].L(w,b)=−​m​​1​​∑​i=1​m​​[y​i​​ln​p​^​​​i​​+(1−y​i​​)ln(1−​p​^​​​i​​)].

训练时同样通过梯度下降（或其它优化器）使 L\mathcal{L}L 最小化。

loss 函数的梯度

与 线性回归模型 一样，先对 loss 求参数梯度 ∇L\nabla\mathcal{L}∇L，再沿负梯度方向更新 w\mathbf{w}w 和 bbb。

记第 iii 个样本的线性打分为 zi=w⊤xi+bz_i = \mathbf{w}^\top \mathbf{x}_i + bz​i​​=w​⊤​​x​i​​+b，预测概率为 p^i=sigmoid(zi)\hat{p}_i = \mathrm{sigmoid}(z_i)​p​^​​​i​​=sigmoid(z​i​​)。定义 概率残差

δi=p^i−yi.\delta_i = \hat{p}_i - y_i.δ​i​​=​p​^​​​i​​−y​i​​.

对单个样本的交叉熵 ℓi=−[yilnp^i+(1−yi)ln(1−p^i)]\ell_i = -\Bigl[y_i \ln \hat{p}_i + (1 - y_i)\ln(1 - \hat{p}_i)\Bigr]ℓ​i​​=−[y​i​​ln​p​^​​​i​​+(1−y​i​​)ln(1−​p​^​​​i​​)]，由链式法则（sigmoid 与 log 相消）可得

∂ℓi∂zi=p^i−yi=δi.\frac{\partial \ell_i}{\partial z_i} = \hat{p}_i - y_i = \delta_i.​∂z​i​​​​∂ℓ​i​​​​=​p​^​​​i​​−y​i​​=δ​i​​.

又 ∂zi∂wj=xij\dfrac{\partial z_i}{\partial w_j} = x_{ij}​∂w​j​​​​∂z​i​​​​=x​ij​​、∂zi∂b=1\dfrac{\partial z_i}{\partial b} = 1​∂b​​∂z​i​​​​=1，对全部 mmm 个样本求平均后，有

∂L∂wj=1m∑i=1mδixij,\frac{\partial \mathcal{L}}{\partial w_j} = \frac{1}{m}\sum_{i=1}^{m} \delta_i\, x_{ij},​∂w​j​​​​∂L​​=​m​​1​​∑​i=1​m​​δ​i​​x​ij​​,

∂L∂b=1m∑i=1mδi.\frac{\partial \mathcal{L}}{\partial b} = \frac{1}{m}\sum_{i=1}^{m} \delta_i.​∂b​​∂L​​=​m​​1​​∑​i=1​m​​δ​i​​.

写成向量形式，权重梯度为

∇wL(w,b)=1m∑i=1m(p^i−yi)xi.\nabla_{\mathbf{w}} \mathcal{L}(\mathbf{w}, b) = \frac{1}{m}\sum_{i=1}^{m} (\hat{p}_i - y_i)\,\mathbf{x}_i.∇​w​​L(w,b)=​m​​1​​∑​i=1​m​​(​p​^​​​i​​−y​i​​)x​i​​.

梯度（按 (w,b)(\mathbf{w}, b)(w,b) 顺序排成向量）为

∇L(w,b)=(∇wL∂L∂b).\nabla \mathcal{L}(\mathbf{w}, b) = \begin{pmatrix} \nabla_{\mathbf{w}} \mathcal{L} \\ \dfrac{\partial \mathcal{L}}{\partial b} \end{pmatrix}.∇L(w,b)=(​∇​w​​L​​∂b​​∂L​​​​).

与线性回归对比：MSE 的梯度里是 数值残差 yi−y^iy_i - \hat{y}_iy​i​​−​y​^​​​i​​；逻辑回归里是 概率残差 p^i−yi\hat{p}_i - y_i​p​^​​​i​​−y​i​​（或 yi−p^iy_i - \hat{p}_iy​i​​−​p​^​​​i​​，差一个负号），形式同样简洁。

反向更新 w\mathbf{w}w 和 bbb

先 前向 计算 zi→p^i→Lz_i \to \hat{p}_i \to \mathcal{L}z​i​​→​p​^​​​i​​→L，再按上节公式把误差从 loss 端传回到参数端，用 梯度下降 更新一次。

取 学习率 η>0\eta > 0η>0，更新规则为

w←w−η∇wL,b←b−η∂L∂b.\mathbf{w} \leftarrow \mathbf{w} - \eta\,\nabla_{\mathbf{w}} \mathcal{L}, \qquad b \leftarrow b - \eta\,\frac{\partial \mathcal{L}}{\partial b}.w←w−η∇​w​​L,b←b−η​∂b​​∂L​​.

代入偏导，即

w←w−ηm∑i=1m(p^i−yi)xi,b←b−ηm∑i=1m(p^i−yi).\mathbf{w} \leftarrow \mathbf{w} - \frac{\eta}{m}\sum_{i=1}^{m} (\hat{p}_i - y_i)\,\mathbf{x}_i, \qquad b \leftarrow b - \frac{\eta}{m}\sum_{i=1}^{m} (\hat{p}_i - y_i).w←w−​m​​η​​∑​i=1​m​​(​p​^​​​i​​−y​i​​)x​i​​,b←b−​m​​η​​∑​i=1​m​​(​p​^​​​i​​−y​i​​).

等价地也可写成

w←w+ηm∑i=1m(yi−p^i)xi,b←b+ηm∑i=1m(yi−p^i).\mathbf{w} \leftarrow \mathbf{w} + \frac{\eta}{m}\sum_{i=1}^{m} (y_i - \hat{p}_i)\,\mathbf{x}_i, \qquad b \leftarrow b + \frac{\eta}{m}\sum_{i=1}^{m} (y_i - \hat{p}_i).w←w+​m​​η​​∑​i=1​m​​(y​i​​−​p​^​​​i​​)x​i​​,b←b+​m​​η​​∑​i=1​m​​(y​i​​−​p​^​​​i​​).

多轮迭代后，L\mathcal{L}L 会逐渐下降；交叉熵对 w,b\mathbf{w}, bw,b 是凸函数，在合适的学习率下可收敛到全局最优。
