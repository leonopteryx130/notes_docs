# 全连接神经网络

### 神经元和逻辑回归模型

**逻辑回归**：输入 $$\mathbf{x}$$ → 乘以权重 $$\mathbf{W}$$ → 加上偏置 $$b$$ → 送入 Sigmoid（激活函数）→ 输出概率 $$y$$。

**单个神经元**：输入 $$\mathbf{x}$$ → 线性变换（$$\mathbf{W}\mathbf{x} + b$$）→ 激活函数（Sigmoid）→ 输出 $$y$$。

两者计算流程相同，逻辑回归就是一个 Sigmoid 神经元。

单个神经元表达能力有限。把多个神经元并排组成**隐藏层**，对输入做非线性变换，提取更复杂的特征。隐藏层位于输入层和输出层之间，层数越多，网络越深。

**隐藏层的意义**：原始输入 $$\mathbf{x}$$ 往往是低层、原始的特征（如图片的像素值），不一定能直接用于分类。隐藏层的作用，就是一层层把原始特征**变换成更有用的高层特征**——浅层可能学到边缘、颜色等简单模式，深层可能学到形状、纹理等更抽象的模式。输出层再基于这些提取好的特征做最终判断。

例如识别猫：$$\mathbf{x}$$（像素）→ 隐藏层（边缘、轮廓）→ 隐藏层（耳朵、眼睛）→ 输出层（是猫 / 不是猫）。

### 从神经元到隐藏层

隐藏层的每个神经元负责检测一种特征。输入 $$\mathbf{x}$$ 经线性变换后，激活函数（如 ReLU）本质上是在做**二分类：这个特征有没有被提取出来**——激活（输出 > 0）表示「从这份数据里检测到了这个特征」，未激活（输出 = 0）表示「没有」。

多个神经元并行，各自回答一个「有没有某特征」的问题，激活结果拼成向量 $$\mathbf{h}$$，就是这一层提取到的特征集合。

### 从隐藏层到输出层

隐藏层输出的 $$\mathbf{h}$$ 是一组特征的激活状态（哪些特征被检测到了）。输出层把这些激活状态当作**新的输入**，再做一次回归或分类，得到最终结果 $$\mathbf{y}$$。

- **回归任务**：直接输出连续值（如房价预测）
- **分类任务**：二分类用 Sigmoid 输出概率；多分类用 Softmax 输出各类别概率

可以理解为：隐藏层负责「提取特征」，输出层负责「根据特征做判断」——和逻辑回归一样，只是输入从原始 $$\mathbf{x}$$ 换成了隐藏层提取好的 $$\mathbf{h}$$。

### 梯度回传

从输入层到隐藏层，每个神经元本质上都是逻辑回归的公式（线性变换 + 激活函数）。这一层的输出作为下一层的输入，再套一层同样的公式——所以**一层全连接网络，就是逻辑回归外面再套一层复合函数**；多层隐藏层，就是多套几层复合函数：

$$\hat{y} = f^{(L)} \circ f^{(L-1)} \circ \cdots \circ f^{(1)}(\mathbf{x})$$

训练时要更新每一层的权重，但 loss 只定义在最终输出 $$\hat{y}$$ 上。**梯度回传**（反向传播）就是用**链式法则**，把 loss 对参数的梯度，从输出层一层层往回传：

1. 先算输出层的梯度（和单层逻辑回归一样）
2. 再传给前一层：这一层的梯度 = 后一层传回来的梯度 × 本层激活函数的导数
3. 逐层往前，直到输入层

每一层收到的都是「后面层需要我调整多少」，据此更新自己的权重。层数越多，复合越深，链式法则乘的项越多，早期层梯度可能变小（梯度消失）。
