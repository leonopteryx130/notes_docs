# 直线的光栅化算法

### DDA 数值微分算法

DDA（Digital Differential Analyzer）算法是一种简单直观的直线光栅化方法。已知直线上任意两点，直线可用斜截式表示：

$$
y = kx + b
$$

其中 $$k$$ 为斜率。根据 $$|k|$$ 的大小，直线的**主行进方向**不同：

- 若 $$|k| \le 1$$，$$x$$ 方向变化更快，主行进方向为 $$x$$ 轴；
- 若 $$|k| > 1$$，$$y$$ 方向变化更快，主行进方向为 $$y$$ 轴。

具体画线规则如下：

1. 当 $$|k| \le 1$$ 时，从起点出发，每次 $$x = x + 1$$，$$y = y + k$$，再将 $$y$$ 四舍五入取整，得到的 $$(x, y)$$ 即为应点亮的像素。

2. 当 $$|k| > 1$$ 时，从起点出发，每次 $$y = y + 1$$，$$x = x + 1/k$$，再将 $$x$$ 四舍五入取整，得到的 $$(x, y)$$ 即为应点亮的像素。

这样按主轴逐格推进，并用斜率累加另一轴坐标，即可在像素网格上近似画出整条直线。


### 中点Bresenham算法

中点 Bresenham 算法用**整数运算**完成直线光栅化，避免 DDA 中的浮点累加与四舍五入。这里只讨论 $$0 < k < 1$$ 的情况：每次 $$x$$ 加 1，只需在右方像素 $$E(x+1, y)$$ 与右上方像素 $$NE(x+1, y+1)$$ 之间二选一。

#### 核心思想

设直线隐式方程为：

$$
F(x, y) = ax + by + c = 0
$$

其中 $$a = \Delta y$$，$$b = -\Delta x$$，$$\Delta x = x_1 - x_0$$，$$\Delta y = y_1 - y_0$$。对落在直线上的点有 $$F = 0$$；直线上方 $$F < 0$$，下方 $$F > 0$$（约定因 $$b < 0$$ 而来）。

当前已选中像素 $$(x_p, y_p)$$ 时，下一步候选为 $$E$$ 与 $$NE$$。取二者中点：

$$
M = \bigl(x_p + 1,\ y_p + \tfrac{1}{2}\bigr)
$$

用 $$F(M)$$ 判断理想直线相对中点的位置，即可决定选哪个像素：

1. 若直线在中点**上方**（$$F(M) < 0$$），选 $$NE$$；
2. 若直线在中点**下方**（$$F(M) > 0$$），选 $$E$$；
3. 若直线恰好过中点（$$F(M) = 0$$），任选其一即可。

#### 决策参数与递推

直接算 $$F(M)$$ 仍含分数。定义决策参数 $$d = 2F(M)$$，则只需比较 $$d$$ 的符号，且全程可用整数更新。

初始（从起点 $$(x_0, y_0)$$ 出发）：

$$
d_0 = 2\Delta y - \Delta x
$$

每步 $$x \leftarrow x + 1$$，再按 $$d$$ 更新：

- 若 $$d < 0$$：选 $$E$$，$$d \leftarrow d + 2\Delta y$$；
- 若 $$d \ge 0$$：选 $$NE$$，$$y \leftarrow y + 1$$，$$d \leftarrow d + 2(\Delta y - \Delta x)$$。

重复直到到达终点。全程仅加减与比较，无浮点乘除，这是相对 DDA 的主要优点。
