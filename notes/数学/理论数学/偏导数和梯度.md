# 偏导数和梯度

### 偏导数

设多元函数 $$f(x_1, x_2, \ldots, x_n)$$ 在某点附近对各坐标变量可偏导。**对第 $$i$$ 个变量的偏导数**记为

$$\frac{\partial f}{\partial x_i},$$

它表示：**其余变量都固定不动，只让 $$x_i$$ 变化时**，$$f$$ 关于 $$x_i$$ 的变化率。在一点处求出后，**偏导数是一个数（标量）**。

### 梯度

在同一函数、同一点处，把所有偏导数按坐标顺序排成一个向量，称为 **梯度**（gradient），常记为 $$\nabla f$$ 

$$\nabla f = \left(\frac{\partial f}{\partial x_1},\frac{\partial f}{\partial x_2},\ldots,\frac{\partial f}{\partial x_n}\right).$$

也就是说：**梯度向量的第 $$i$$ 个分量，就是对 $$x_i$$ 的偏导数**。

对二元函数 $$f(x,y)$$，常写成

$$\nabla f = \left(\frac{\partial f}{\partial x},\frac{\partial f}{\partial y}\right).$$

### 偏导数与梯度的关系


|     | 偏导数 $$\dfrac{\partial f}{\partial x_i}$$ | 梯度 $$\nabla f$$      |
| --- | ---------------------------------------- | -------------------- |
| 类型  | 标量（一个数）                                  | 向量（有序数组）             |
| 含义  | 沿第 $$i$$ 条坐标轴方向的瞬时变化率                    | 它的方向是函数在该点增加最快的方向；它的模（大小）就是该方向上的最大方向导数。 |


**梯度向量的方向**是函数在该点 **上升最快** 的方向；**梯度的模长**等于该方向上的 **最大方向导数**（沿该方向单位步长下 $$f$$ 的瞬时增长率）。

### 几何与直观

- **求偏导**：相当于问「**只**沿某一条坐标轴挪一小步，函数值变多少？」
- **求梯度**：相当于问「往 **哪个方向** 挪一小步，函数值 **上升最快**？这个最快的变化率是多少？」

二者紧密相连：偏导数是「各坐标轴方向」上的局部信息；梯度把这些信息合成一个向量，直接给出最陡上升方向及其陡峭程度。
