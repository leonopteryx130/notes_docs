# 复合函数求导

### 什么是复合函数？

若 $$y$$ 是 $$u$$ 的函数，$$u$$ 又是 $$x$$ 的函数，即

$$y = f(u), \quad u = g(x),$$

则 $$y$$ 通过中间变量 $$u$$ 成为 $$x$$ 的函数，记作

$$y = f(g(x)) = (f \circ g)(x),$$

称为 **$$f$$ 与 $$g$$ 的复合函数**，$$u$$ 常称为 **中间变量**。复合函数把「外层」$$f$$ 与「内层」$$g$$ 嵌套在一起；求导时要把两层的变化率按顺序乘起来，这就是链式法则。

### 链式法则

设 $$g$$ 在 $$x$$ 处可导，$$f$$ 在 $$u = g(x)$$ 处可导，则复合函数 $$f \circ g$$ 在 $$x$$ 处可导，且

$$\frac{\mathrm{d}y}{\mathrm{d}x} = \frac{\mathrm{d}y}{\mathrm{d}u} \cdot \frac{\mathrm{d}u}{\mathrm{d}x} = f'(g(x)) \cdot g'(x).$$

**莱布尼茨记法**下常写成：若 $$y = f(u)$$、$$u = g(x)$$，则

$$\frac{\mathrm{d}y}{\mathrm{d}x} = \frac{\mathrm{d}y}{\mathrm{d}u} \cdot \frac{\mathrm{d}u}{\mathrm{d}x}.$$

**多重复合**：若 $$y = f(u)$$、$$u = g(v)$$、$$v = h(x)$$，则

$$\frac{\mathrm{d}y}{\mathrm{d}x} = \frac{\mathrm{d}y}{\mathrm{d}u} \cdot \frac{\mathrm{d}u}{\mathrm{d}v} \cdot \frac{\mathrm{d}v}{\mathrm{d}x}.$$

### 举例

**例 1（幂与内层线性）** 设 $$y = (3x + 1)^5$$，令 $$u = 3x + 1$$，则 $$y = u^5$$，

$$\frac{\mathrm{d}y}{\mathrm{d}x} = 5u^4 \cdot 3 = 15(3x + 1)^4.$$

**例 2（指数）** 设 $$y = \mathrm{e}^{x^2}$$，令 $$u = x^2$$，则 $$y = \mathrm{e}^u$$，

$$\frac{\mathrm{d}y}{\mathrm{d}x} = \mathrm{e}^u \cdot 2x = 2x\,\mathrm{e}^{x^2}.$$

**例 3（对数）** 设 $$y = \ln(2x + 3)$$，令 $$u = 2x + 3$$，则 $$y = \ln u$$，

$$\frac{\mathrm{d}y}{\mathrm{d}x} = \frac{1}{u} \cdot 2 = \frac{2}{2x + 3}.$$

**例 4（三角函数）** 设 $$y = \sin(5x)$$，

$$\frac{\mathrm{d}y}{\mathrm{d}x} = \cos(5x) \cdot 5 = 5\cos(5x).$$

以上各例都体现了同一结构：先对外层函数关于中间变量求导，再乘以内层函数关于自变量的导数。
