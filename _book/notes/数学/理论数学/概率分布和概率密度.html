概率分布和概率密度

概率密度

从离散到连续

离散随机变量用概率质量函数（PMF）描述：对每个取值 xxx，P(X=x)P(X = x)P(X=x) 就是概率本身，且所有取值概率之和为 1。

连续随机变量则不同：对任意实数 xxx，单点概率恒为 0：

P(X=x)=0P(X = x) = 0P(X=x)=0

因此不能再用「某点的概率」来描述分布，而要用概率密度函数（PDF）刻画「落在某个区间内的概率有多大」。

定义

设连续随机变量 XXX 的分布函数为 F(x)=P(X≤x)F(x) = P(X \le x)F(x)=P(X≤x)。若存在非负函数 f(x)f(x)f(x)，使得对任意实数 a<ba < ba<b 都有

P(a<X≤b)=∫abf(x)dxP(a < X \le b) = \int_a^b f(x)\,\mathrm{d}xP(a<X≤b)=∫​a​b​​f(x)dx

则称 f(x)f(x)f(x) 为 XXX 的概率密度函数（probability density function，缩写 PDF）；有时也记为 fX(x)f_X(x)f​X​​(x)，强调它属于随机变量 XXX。

直观理解：f(x)f(x)f(x) 在 xxx 点的值，正比于随机变量落在「xxx 点附近一个无穷小邻域内」的概率。用微积分语言说，当 Δx\Delta xΔx 非常小时：

P(x<X≤x+Δx)≈f(x)ΔxP(x < X \le x + \Delta x) \approx f(x)\,\Delta xP(x<X≤x+Δx)≈f(x)Δx

等价地，密度也可写成极限形式（在 FFF 可导的点处）：

f(x)=limΔx→0+P(x<X≤x+Δx)Δx=F′(x)f(x) = \lim_{\Delta x \to 0^+} \frac{P(x < X \le x + \Delta x)}{\Delta x} = F'(x)f(x)=lim​Δx→0​+​​​​​Δx​​P(x<X≤x+Δx)​​=F​′​​(x)

概率分布

概率分布描述随机变量「取各个值（或落在各个区间）的可能性有多大」。

  • 离散型：用概率质量函数（PMF）P(X=x)P(X = x)P(X=x) 列出每个取值的概率，所有概率之和为 1。
  • 连续型：单点概率为 0，用概率密度函数（PDF）f(x)f(x)f(x) 描述；区间概率由积分给出（见上文）。

两种情形都能用分布函数（CDF）统一表示：

F(x)=P(X≤x)F(x) = P(X \le x)F(x)=P(X≤x)

含义：随机变量 XXX 不超过 xxx 的概率。CDF 单调不减，且 F(−∞)=0F(-\infty)=0F(−∞)=0、F(+∞)=1F(+\infty)=1F(+∞)=1。

记法：若 XXX 服从某分布，常写作 X∼某分布名X \sim \text{某分布名}X∼某分布名（如 X∼N(μ,σ2)X \sim \mathcal{N}(\mu, \sigma^2)X∼N(μ,σ​2​​) 表示正态分布）。

对比

直观理解（用考试分数举例）

假设全班同学数学考试成绩在 0 到 100 分之间，且成绩是连续型随机变量。

  • 若问：「恰好考了 60 分」的概率是多少？
    答案是 0。连续型变量取任意单个值的概率恒为 0，因此不能靠「某点的概率」来描述分布。

  • 若问：「考了 59 到 61 分之间」的概率是多少？
    这才是 PDF 的用武之地：P(59<X≤61)=∫5961f(x)dxP(59 < X \le 61) = \int_{59}^{61} f(x)\,\mathrm{d}xP(59<X≤61)=∫​59​61​​f(x)dx。
    密度 f(60)f(60)f(60) 本身不是概率，而是「60 分附近单位区间内概率有多大」的相对快慢——f(60)f(60)f(60) 越大，说明成绩落在 60 分附近的概率相对越高。

  • 若问：「考了 60 分及以下」的概率是多少？
    比如算出来是 30%，那这个 30% 就是分布函数在 x=60x=60x=60 时的值，记作 F(60)=0.3F(60) = 0.3F(60)=0.3。

随着 xxx 从 0 增大到 100，F(x)F(x)F(x) 会从 0 一直累加到 1。所以分布函数永远是一条从 0 爬到 1 的单调不减曲线。
