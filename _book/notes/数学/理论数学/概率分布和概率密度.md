# 概率分布和概率密度

### 概率密度

#### 从离散到连续

离散随机变量用**概率质量函数**（PMF）描述：对每个取值 $$x$$，$$P(X = x)$$ 就是概率本身，且所有取值概率之和为 1。

连续随机变量则不同：对任意实数 $$x$$，单点概率恒为 0：

$$P(X = x) = 0$$

因此不能再用「某点的概率」来描述分布，而要用**概率密度函数**（PDF）刻画「落在某个区间内的概率有多大」。

---

#### 定义

设连续随机变量 $$X$$ 的分布函数为 $$F(x) = P(X \le x)$$。若存在非负函数 $$f(x)$$，使得对任意实数 $$a < b$$ 都有

$$P(a < X \le b) = \int_a^b f(x)\,\mathrm{d}x$$

则称 $$f(x)$$ 为 $$X$$ 的**概率密度函数**（probability density function，缩写 **PDF**）；有时也记为 $$f_X(x)$$，强调它属于随机变量 $$X$$。

**直观理解**：$$f(x)$$ 在 $$x$$ 点的值，正比于随机变量落在「$$x$$ 点附近一个无穷小邻域内」的概率。用微积分语言说，当 $$\Delta x$$ 非常小时：

$$P(x < X \le x + \Delta x) \approx f(x)\,\Delta x$$

等价地，密度也可写成极限形式（在 $$F$$ 可导的点处）：

$$f(x) = \lim_{\Delta x \to 0^+} \frac{P(x < X \le x + \Delta x)}{\Delta x} = F'(x)$$

---

### 概率分布

**概率分布**描述随机变量「取各个值（或落在各个区间）的可能性有多大」。

- **离散型**：用**概率质量函数**（PMF）$$P(X = x)$$ 列出每个取值的概率，所有概率之和为 1。
- **连续型**：单点概率为 0，用**概率密度函数**（PDF）$$f(x)$$ 描述；区间概率由积分给出（见上文）。

两种情形都能用**分布函数**（CDF）统一表示：

$$F(x) = P(X \le x)$$

含义：随机变量 $$X$$ 不超过 $$x$$ 的概率。CDF 单调不减，且 $$F(-\infty)=0$$、$$F(+\infty)=1$$。

记法：若 $$X$$ 服从某分布，常写作 $$X \sim \text{某分布名}$$（如 $$X \sim \mathcal{N}(\mu, \sigma^2)$$ 表示正态分布）。

### 对比

#### 直观理解（用考试分数举例）

假设全班同学数学考试成绩在 0 到 100 分之间，且成绩是**连续型**随机变量。

- 若问：「**恰好**考了 60 分」的概率是多少？  
  答案是 **0**。连续型变量取任意单个值的概率恒为 0，因此不能靠「某点的概率」来描述分布。

- 若问：「考了 **59 到 61 分之间**」的概率是多少？  
  这才是 PDF 的用武之地：$$P(59 < X \le 61) = \int_{59}^{61} f(x)\,\mathrm{d}x$$。  
  密度 $$f(60)$$ 本身**不是概率**，而是「60 分附近单位区间内概率有多大」的**相对快慢**——$$f(60)$$ 越大，说明成绩落在 60 分附近的概率相对越高。

- 若问：「考了 **60 分及以下**」的概率是多少？  
  比如算出来是 30%，那这个 30% 就是**分布函数**在 $$x=60$$ 时的值，记作 $$F(60) = 0.3$$。

随着 $$x$$ 从 0 增大到 100，$$F(x)$$ 会从 0 一直累加到 1。所以分布函数永远是一条从 0 爬到 1 的**单调不减**曲线。

---

#### 表格对比

| | **概率密度函数** $$f(x)$$（PDF） | **分布函数** $$F(x)$$（CDF） |
|---|---|---|
| **含义** | 描述随机变量落在 $$x$$ **附近**的概率有多「密」 | 描述随机变量 **不超过** $$x$$ 的累积概率 |
| **取值** | 非负，$$f(x) \ge 0$$；**不是概率**，可以大于 1 | 在 $$[0, 1]$$ 之间，**本身就是概率** |
| **单点** | $$f(x)$$ 表示密度；$$P(X = x) = 0$$（连续型） | $$F(x) = P(X \le x)$$，直接可读 |
| **区间概率** | $$P(a < X \le b) = \int_a^b f(x)\,\mathrm{d}x$$ | $$P(a < X \le b) = F(b) - F(a)$$ |
| **归一化** | $$\int_{-\infty}^{+\infty} f(x)\,\mathrm{d}x = 1$$ | $$F(-\infty) = 0$$，$$F(+\infty) = 1$$ |
| **单调性** | 无单调要求（可以上下波动） | 单调不减 |
| **关系** | 若 $$F$$ 可导，则 $$f(x) = F'(x)$$ | $$F(x) = \int_{-\infty}^{x} f(t)\,\mathrm{d}t$$ |

**一句话**：PDF 告诉你「哪里更可能」；CDF 告诉你「到某个位置为止已经累积了多少概率」。两者通过**积分 / 求导**互相转化。
