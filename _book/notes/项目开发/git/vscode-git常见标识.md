# vscode-git常见标识

#### **M modified**

你已经在git中添加过该文件，然后你对这个文件进行了修改，但是没有push到远程仓库
<div align="center">
    <img src=./vscode-git常见标识2.png width=30% />
</div>


#### **U untracked**
你在本地新建了这个文件，还未commit到git中，就会标记U
<div align="center">
    <img src=./vscode-git常见标识1.png width=30% />
</div>

#### **D delete**
你删除了这个文件，git会记录下这个状态

#### **6,U**
表示有6个错误且untracked
