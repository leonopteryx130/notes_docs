- **标签分类**

**硬标签**是非 0 即 1 的确定答案，二分类是 0 或 1，多分类用 **one-hot**（如「猫」是 `[1, 0, 0]`，只有一个位置是 1）。**软标签**是各类别的概率，如 `[0.6, 0.3, 0.1]`，常见于**标签平滑**、**知识蒸馏**。**多热标签**允许一个样本同时属于多个类，如图片同时有「猫」和「室内」→ `[1, 0, 1, 0]`，和 **one-hot** 的区别是各类可以并存。

- **全连接层**

全连接层是一个**计算单元**，做线性变换：

$$y = Wx + b$$

它仅作线性变换，区别于**全连接神经网络**：没有激活函数，也没有隐藏层。

- **隐藏层**

隐藏层是一个**位置概念**：指神经网络中，除输入层和输出层之外的中间层。全连接神经网络里，一个隐藏层通常由「全连接层（线性变换）+ 激活函数（非线性变换）」共同组成。

- **ground truth**

机器学习里的 ground truth 一般指数据集的的真实标签
