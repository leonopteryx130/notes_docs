# 逻辑回归模型

**逻辑回归**用于二分类：给定特征 $$\mathbf{x}$$，预测样本属于正类（$$y=1$$）的概率。

与 [线性回归模型](./线性回归模型.md) 一样，先算线性打分，再经激活函数得到概率，最后用交叉熵作为 loss，通过梯度下降更新参数。

---

### 1. 线性计算

对单个样本，先算线性组合：

$$z = \mathbf{w}^\top \mathbf{x} + b$$

此时 $$z$$ 称为 **Logits**（对数几率），取值范围是 $$(-\infty, +\infty)$$。

---

### 2. 激活（逆 logit）

把 Logits 映射到概率区间 $$[0, 1]$$，用 **Sigmoid** 函数：

$$a = \mathrm{Sigmoid}(z) = \frac{1}{1 + e^{-z}}$$

其中 $$a$$ 是预测概率 $$\hat{p}$$。Sigmoid 是 [Logit 函数](../理论数学/概率，几率，对数几率.md) 的反函数，即 $$\mathrm{logit}(a) = z$$。

---

### 3. 计算 Loss

真实标签 $$y \in \{0, 1\}$$，单个样本的 **交叉熵损失** 为：

$$\ell = -\bigl[y \ln a + (1 - y)\ln(1 - a)\bigr]$$

$$m$$ 个样本取平均：

$$\mathcal{L} = -\frac{1}{m}\sum_{i=1}^{m} \Bigl[y_i \ln a_i + (1 - y_i)\ln(1 - a_i)\Bigr]$$

---

### 4. 链式求导 + 偏导数

训练目标：求 loss 对参数的梯度 $$\nabla\mathcal{L}$$，再沿负梯度方向更新。

记第 $$i$$ 个样本：$$z_i = \mathbf{w}^\top \mathbf{x}_i + b$$，$$a_i = \mathrm{Sigmoid}(z_i)$$。

**链式法则**（从 loss 传到 $$z_i$$）：

$$\frac{\partial \ell_i}{\partial z_i}
= \frac{\partial \ell_i}{\partial a_i} \cdot \frac{\partial a_i}{\partial z_i}
= a_i - y_i$$

（sigmoid 与 log 相消，结果很简洁。）

**再对参数求偏导**（$$z_i = \mathbf{w}^\top \mathbf{x}_i + b$$）：

$$\frac{\partial z_i}{\partial w_j} = x_{ij}, \qquad \frac{\partial z_i}{\partial b} = 1$$

对全部样本求平均后：

$$\frac{\partial \mathcal{L}}{\partial w_j}
= \frac{1}{m}\sum_{i=1}^{m} (a_i - y_i)\, x_{ij},$$

$$\frac{\partial \mathcal{L}}{\partial b}
= \frac{1}{m}\sum_{i=1}^{m} (a_i - y_i).$$

向量形式：

$$\nabla_{\mathbf{w}} \mathcal{L} = \frac{1}{m}\sum_{i=1}^{m} (a_i - y_i)\,\mathbf{x}_i.$$

---

### 5. 参数更新

取学习率 $$\eta > 0$$，梯度下降更新：

$$\mathbf{w} \leftarrow \mathbf{w} - \eta\,\nabla_{\mathbf{w}} \mathcal{L},
\qquad
b \leftarrow b - \eta\,\frac{\partial \mathcal{L}}{\partial b}$$

代入偏导，即：

$$\mathbf{w} \leftarrow \mathbf{w} - \frac{\eta}{m}\sum_{i=1}^{m} (a_i - y_i)\,\mathbf{x}_i,
\qquad
b \leftarrow b - \frac{\eta}{m}\sum_{i=1}^{m} (a_i - y_i).$$

重复「前向计算 $$z \to a \to \mathcal{L}$$ → 求梯度 → 更新参数」，直到 loss 收敛。

---

### Q&A：为什么不用「平方误差」？

想象一下，你在预测一个客户是否会违约（$$1$$ = 违约，$$0$$ = 不违约）。模型预测违约概率是 $$0.99$$，结果他没违约（真实标签是 $$0$$）。

- **如果用平方误差（MSE）**：损失是 $$(0.99 - 0)^2 = 0.9801$$。
- **如果用逻辑回归的损失（Log Loss）**：损失是 $$-\ln(1 - 0.99) = -\ln(0.01) \approx 4.61$$。

区别在于：逻辑回归的损失函数对「过分自信的错误」惩罚极重。当你非常肯定地（概率 $$0.99$$）预测他会违约，结果却错了，这个损失会趋向无穷大。这迫使模型不敢轻易「拍脑袋」，必须给出谨慎的概率估计。
