# 逻辑回归模型

### 理论基础：线性回归模型

**线性回归**用特征的线性组合直接给出预测值。对第 $$i$$ 个样本，模型为

$$\hat{y}_i = \mathbf{w}^\top \mathbf{x}_i + b,$$

其中 $$\mathbf{x}_i$$ 为特征向量，$$\mathbf{w}$$ 为权重，$$b$$ 为偏置。训练时在 loss 函数（如 MSE）下通过梯度下降求出 $$\mathbf{w}$$ 和 $$b$$，使预测值 $$\hat{y}_i$$ 尽量接近真实标签 $$y_i$$。具体定义、loss 与梯度推导见 [线性回归模型](./线性回归模型.md)。

逻辑回归与线性回归**共享同一套「线性打分」结构**：都是先算一个实数

$$z_i = \mathbf{w}^\top \mathbf{x}_i + b,$$

再把这个 $$z_i$$ 解释成某种「预测结果」。

### 什么是逻辑回归模型？

**逻辑回归（Logistic Regression）** 主要用来做 **二分类**：给定特征 $$\mathbf{x}$$，估计样本属于正类（$$y=1$$）的概率 $$p$$。

**为什么不能直接把线性回归用于二分类？** 二分类标签常记为 $$y \in 0, 1$$，我们真正想要的是「属于正类的**概率**」$$p \in [0, 1]$$。线性组合 $$\mathbf{w}^\top \mathbf{x} + b$$ 可以取任意实数，既可能小于 $$0$$，也可能大于 $$1$$，无法直接当作概率；若仍用 MSE 去拟合 $$0/1$$ 标签，拟合面也不如专门的概率模型合理。逻辑回归的做法是：**保留线性回归的打分形式，再增加一步把实数映射到 $$[0, 1]$$ 的变换**——这一步通常借助 [概率，几率，对数几率](../理论数学/概率，几率，对数几率.md) 中的 **sigmoid** 与 **logit**。

对第 $$i$$ 个样本，先算线性打分 $$z_i = \mathbf{w}^\top \mathbf{x}_i + b$$，再用 **sigmoid** 函数把 $$z_i$$ 压到 $$[0, 1]$$，得到属于正类的预测概率：

$$\hat{p}_i = \mathrm{sigmoid}(z_i) = \frac{1}{1 + e^{-z_i}}.$$

等价地，模型在对数几率（logit）空间做线性回归：

$$\text{logit}(\hat{p}_i) = \ln\frac{\hat{p}_i}{1 - \hat{p}_i} = z_i = \mathbf{w}^\top \mathbf{x}_i + b.$$

因此逻辑回归的全流程可以概括为：**线性打分 → sigmoid → 概率**。

### loss 函数推导

线性回归的 MSE 为（见 [线性回归模型](./线性回归模型.md)）：

$$\mathcal{L}(\mathbf{w}, b) = \frac{1}{m}\sum_{i=1}^{m} \bigl(y_i - \hat{y}_i\bigr)^2, \qquad \hat{y}_i = \mathbf{w}^\top \mathbf{x}_i + b.$$

逻辑回归保留同样的线性打分，但把 $$\hat{y}_i$$ 换成 [对数几率](../理论数学/概率，几率，对数几率.md) $$\text{logit}(\hat{p}_i)$$：

$$\text{logit}(\hat{p}_i) = \ln\frac{\hat{p}_i}{1 - \hat{p}_i} = \mathbf{w}^\top \mathbf{x}_i + b, \qquad \hat{p}_i = \frac{1}{1 + e^{-(\mathbf{w}^\top \mathbf{x}_i + b)}}.$$

将 $$\hat{p}_i$$ 代入二分类的 **交叉熵损失**（也称 **对数损失 / log loss**），即得逻辑回归的 loss：

$$\mathcal{L}(\mathbf{w}, b) = -\frac{1}{m}\sum_{i=1}^{m} \Bigl[y_i \ln \hat{p}_i + (1 - y_i)\ln(1 - \hat{p}_i)\Bigr].$$

训练时同样通过梯度下降（或其它优化器）使 $$\mathcal{L}$$ 最小化。

### loss 函数的梯度

与 [线性回归模型](./线性回归模型.md) 一样，先对 **loss 求参数梯度** $$\nabla\mathcal{L}$$，再沿负梯度方向更新 $$\mathbf{w}$$ 和 $$b$$。

记第 $$i$$ 个样本的线性打分为 $$z_i = \mathbf{w}^\top \mathbf{x}_i + b$$，预测概率为 $$\hat{p}_i = \mathrm{sigmoid}(z_i)$$。定义 **概率残差**

$$\delta_i = \hat{p}_i - y_i.$$

对单个样本的交叉熵 $$\ell_i = -\Bigl[y_i \ln \hat{p}_i + (1 - y_i)\ln(1 - \hat{p}_i)\Bigr]$$，由链式法则（sigmoid 与 log 相消）可得

$$\frac{\partial \ell_i}{\partial z_i} = \hat{p}_i - y_i = \delta_i.$$

又 $$\dfrac{\partial z_i}{\partial w_j} = x_{ij}$$、$$\dfrac{\partial z_i}{\partial b} = 1$$，对全部 $$m$$ 个样本求平均后，有

$$\frac{\partial \mathcal{L}}{\partial w_j}
= \frac{1}{m}\sum_{i=1}^{m} \delta_i\, x_{ij},$$

$$\frac{\partial \mathcal{L}}{\partial b}
= \frac{1}{m}\sum_{i=1}^{m} \delta_i.$$

写成向量形式，**权重梯度**为

$$\nabla_{\mathbf{w}} \mathcal{L}(\mathbf{w}, b) = \frac{1}{m}\sum_{i=1}^{m} (\hat{p}_i - y_i)\,\mathbf{x}_i.$$

**梯度**（按 $$(\mathbf{w}, b)$$ 顺序排成向量）为

$$\nabla \mathcal{L}(\mathbf{w}, b) = \begin{pmatrix} \nabla_{\mathbf{w}} \mathcal{L} \\ \dfrac{\partial \mathcal{L}}{\partial b} \end{pmatrix}.$$

与线性回归对比：MSE 的梯度里是 **数值残差** $$y_i - \hat{y}_i$$；逻辑回归里是 **概率残差** $$\hat{p}_i - y_i$$（或 $$y_i - \hat{p}_i$$，差一个负号），形式同样简洁。

### 反向更新 $$\mathbf{w}$$ 和 $$b$$

先 **前向** 计算 $$z_i \to \hat{p}_i \to \mathcal{L}$$，再按上节公式把误差从 loss 端传回到参数端，用 **梯度下降** 更新一次。

取 **学习率** $$\eta > 0$$，更新规则为

$$\mathbf{w} \leftarrow \mathbf{w} - \eta\,\nabla_{\mathbf{w}} \mathcal{L},
\qquad
b \leftarrow b - \eta\,\frac{\partial \mathcal{L}}{\partial b}.$$

代入偏导，即

$$\mathbf{w} \leftarrow \mathbf{w} - \frac{\eta}{m}\sum_{i=1}^{m} (\hat{p}_i - y_i)\,\mathbf{x}_i,
\qquad
b \leftarrow b - \frac{\eta}{m}\sum_{i=1}^{m} (\hat{p}_i - y_i).$$

等价地也可写成

$$\mathbf{w} \leftarrow \mathbf{w} + \frac{\eta}{m}\sum_{i=1}^{m} (y_i - \hat{p}_i)\,\mathbf{x}_i,
\qquad
b \leftarrow b + \frac{\eta}{m}\sum_{i=1}^{m} (y_i - \hat{p}_i).$$

多轮迭代后，$$\mathcal{L}$$ 会逐渐下降；交叉熵对 $$\mathbf{w}, b$$ 是凸函数，在合适的学习率下可收敛到全局最优。
