Bahdanau Attention

Bahdanau Attention

结构图

预备阶段（编码器，图下部）

对输入序列 x1,x2,…,xTx_1, x_2, \ldots, x_Tx​1​​,x​2​​,…,x​T​​，用双向 RNN 在每个位置计算隐藏状态：正向 hi→h_i^{\rightarrow}h​i​→​​ 与反向 hi←h_i^{\leftarrow}h​i​←​​ 拼接为注解向量

hi=[hi→;hi←]h_i = [h_i^{\rightarrow}; h_i^{\leftarrow}]h​i​​=[h​i​→​​;h​i​←​​]

得到整组编码器状态 h1,h2,…,hTh_1, h_2, \ldots, h_Th​1​​,h​2​​,…,h​T​​。这组向量在解码过程中固定不变，相当于把整本书摊开摆在桌面上，供后续每一步查阅。

循环阶段（生成第 ttt 个词，图中部与上部）

1. 取解码器上一时刻状态 st−1s_{t-1}s​t−1​​

输入为解码器上一步隐藏状态 st−1s_{t-1}s​t−1​​。

通常初始状态 s0s_0s​0​​ 由编码器末状态引入，或设为全零向量。

2. 计算能量分数 et,ie_{t,i}e​t,i​​（未归一化）

用 st−1s_{t-1}s​t−1​​ 与每一个编码器状态 hih_ih​i​​ 比对，得到「匹配度」：

et,i=v⊤tanh(Wsst−1+Whhi)e_{t,i} = v^\top \tanh(W_s s_{t-1} + W_h h_i)e​t,i​​=v​⊤​​tanh(W​s​​s​t−1​​+W​h​​h​i​​)

3. Softmax 得到注意力权重 αt,i\alpha_{t,i}α​t,i​​（图中向下的箭头）

将能量分数归一化为概率（各位置权重之和为 1），即图中的 αt,1,αt,2,αt,3,…\alpha_{t,1}, \alpha_{t,2}, \alpha_{t,3}, \ldotsα​t,1​​,α​t,2​​,α​t,3​​,…：

αt,i=exp(et,i)∑j=1Texp(et,j)\alpha_{t,i} = \frac{\exp(e_{t,i})}{\sum_{j=1}^{T} \exp(e_{t,j})}α​t,i​​=​∑​j=1​T​​exp(e​t,j​​)​​exp(e​t,i​​)​​

4. 加权求和得到上下文向量 ctc_tc​t​​

用权重 αt,i\alpha_{t,i}α​t,i​​ 对所有编码器状态 hih_ih​i​​ 做加权求和：

ct=∑i=1Tαt,ihic_t = \sum_{i=1}^{T} \alpha_{t,i}\, h_ic​t​​=∑​i=1​T​​α​t,i​​h​i​​

这是 Bahdanau 注意力最关键的一步：ctc_tc​t​​ 随每一步翻译而变化，不再是固定的单一语义向量。

5. 融合并更新解码器

将上一时刻状态 st−1s_{t-1}s​t−1​​ 与当前上下文 ctc_tc​t​​（以及上一步输出 yt−1y_{t-1}y​t−1​​）一并送入解码器 RNN，得到 sts_ts​t​​，再据此预测 yty_ty​t​​。

关于 additive attention（加性注意力）

最朴素的相似度是点积 e=s⋅he = s \cdot he=s⋅h。它能工作的前提是：解码器隐状态 sss 与编码器隐状态 hhh 必须在同一个向量空间里，且维度相同。

但在机器翻译里，两边往往不在同一空间：

  • 编码器处理源语言（如中文），生成的 hih_ih​i​​ 位于编码器空间；
  • 解码器处理目标语言（如英文），生成的 st−1s_{t-1}s​t−1​​ 位于解码器空间。

直接点积相当于用两种不同坐标系下的向量做内积，语义上不成立。Bahdanau 因此改用 additive attention（加性注意力），先把两边投到同一个对齐空间，再算匹配分：

et,i=v⊤tanh(Wsst−1+Whhi)e_{t,i} = v^\top \tanh(W_s s_{t-1} + W_h h_i)e​t,i​​=v​⊤​​tanh(W​s​​s​t−1​​+W​h​​h​i​​)

各部分的作用：

  1. Wsst−1W_s s_{t-1}W​s​​s​t−1​​：把解码器状态投影到对齐空间（可与编码器空间对齐）。
  2. WhhiW_h h_iW​h​​h​i​​：对编码器状态做一次线性变换，进入同一空间。
  3. Wss+WhhW_s s + W_h hW​s​​s+W​h​​h（相加）：在对齐空间（Alignment Space）里，把两个不同来源的信息融合成一个联合特征向量。「加性」即指这一步，而非直接点积。
  4. tanh\tanhtanh（非线性）：两种语言空间的映射关系极复杂，不是单纯的线性加减；非线性让网络有能力拟合复杂的语义匹配模式。
  5. v⊤v^\topv​⊤​​：tanh\tanhtanh 输出仍是向量，而注意力需要的是标量分数。可学习向量 vvv 负责「读出」该融合向量的能量，压缩成一个对齐得分（Score）。

Ws,Wh,vW_s, W_h, vW​s​​,W​h​​,v 均为可学习参数；sss 与 hhh 维度可以不同（各用自己的投影矩阵）。
