# 概率，几率，对数几率

## 概率

概率很好理解，比如明天下雨的概率是 80%，那么不下雨的概率就是 20%

- 取值范围：$$[0, 1]$$
- 记作 $$P(A)$$ 或 $$p$$

## 几率

**几率（Odds）** 是指事件发生的概率与不发生的概率之比。

- 公式：

$$\text{odds} = \frac{p}{1-p}$$

其中 $$p$$ 为事件发生的概率。
- 例子：
明天下雨的概率 $$p = 0.8$$，那么：

$$\text{odds} = \frac{0.8}{0.2} = 4$$

表示“下雨的几率是 4:1”，即下雨的可能性是不下雨的 4 倍。
- 取值范围：$$[0, +\infty)$$
  - $$\text{odds} = 1$$ 表示发生与不发生的概率相等
  - $$\text{odds} > 1$$ 表示更可能发生
  - $$\text{odds} < 1$$ 表示更不可能发生

**几率**和博彩中**赔率**是一样的

## 对数几率

**对数几率（Log Odds，又称 Logit）** 是几率的自然对数。

- 公式：

$$\text{logit}(p) = \ln\left(\frac{p}{1-p}\right)$$

- 例子：
承接上例，对数几率为：

$$\ln(4) \approx 1.386$$

- 取值范围：$$(-\infty, +\infty)$$
  - 当 $$p = 0.5$$ 时，$$\text{logit}(p) = 0$$
  - 当 $$p > 0.5$$ 时，$$\text{logit}(p) > 0$$
  - 当 $$p < 0.5$$ 时，$$\text{logit}(p) < 0$$

## 为什么要用对数几率？

1. **将概率映射到实数范围**：概率被限制在 $$[0,1]$$，而对数几率可以取任意实数，便于建立回归模型（如逻辑回归）。
2. **对称性**：$$\text{logit}(p)$$ 与 $$\text{logit}(1-p)$$ 互为相反数，利于处理二分类问题。


## 三者关系总结


| 名称           | 公式            | 取值范围     |
| ------------ | ------------- | -------- |
| 概率 (p)       | $$p$$             | $$[0, 1]$$   |
| 几率 (odds)    | $$\dfrac{p}{1-p}$$     | $$[0, +\infty)$$  |
| 对数几率 (logit) | $$\ln\left(\dfrac{p}{1-p}\right)$$ | $$(-\infty, +\infty)$$ |
